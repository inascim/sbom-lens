From src/hooks/useSBOMLibrary.js — getStats() (CURRENT — has mutation bug):
```javascript
const getStats = useCallback(() => {
  const stats = {
    totalSBOMs: sboms.length,
    totalComponents: sboms.reduce((sum, s) => sum + (s.components?.length || 0), 0),
    recentlyModified:
      sboms.length > 0
        ? sboms.sort((a, b) => new Date(b.modifiedAt) - new Date(a.modifiedAt))[0]
        : null,
  };
  return stats;
}, [sboms]);
```
Bug: `sboms.sort(...)` mutates the React state array in place.
Fix: Change to `[...sboms].sort(...)`.
